# Rhythia Unofficial Replay Viewer

This is my web replay viewer for Rhythia `.rhr` files.

I made it so anyone can open and watch replays outside the game client.

## Use It

Live page:
- https://mahvro.github.io/rhythia-unofficial-replay-viewer/

## Quick Guide

1. Open the page.
2. Click the folder button, or drag and drop a `.rhr` file on the canvas.
3. Use the bottom bar to seek or change speed.
4. Use the settings button for visual and marker options.
